# filmio-css

### **Autor:** Erick De Lima

Este projeto tem como objetivo desenvolver uma aplicação web de catálogo de filmes, permitindo que usuários explorem diferentes títulos, visualizem informações sobre os filmes e gerenciem uma conta com cadastro e login.

O frontend da aplicação foi desenvolvido com HTML, CSS e JavaScript. Os dados dos filmes são obtidos por meio da TMDb API, enquanto o backend é simulado utilizando uma API Fake com JSON Server para armazenar informações de usuários e interações básicas do sistema.
